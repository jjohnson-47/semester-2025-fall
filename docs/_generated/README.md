# Historical Generated Evidence

The files in this directory are frozen outputs from the September 2025
orchestration and probe sessions. They document what those sessions observed or
planned; they are not current runtime state and must not be used as a clean
checkout acceptance gate.

In particular, references to a local SQLite database, an
`.orchestration/tracker.json` file, deployment workflows, or the former
dashboard deployment API are historical evidence. Those session-local or
orphaned surfaces were retired on 2026-07-14 under
[`ADR 0005`](../adr/0005-retained-public-archive.md).

Current project posture is recorded in the repository root
[`project.yaml`](../../project.yaml) and [`README.md`](../../README.md).
